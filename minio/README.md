# Minio

A Dagger module to provide a Minio server and use a Minio client.

## Features

| Command                                        | Done |
|------------------------------------------------|------|
| Create and configure a Minio server            | ✅    |
| Create and configure a Minio client            | ✅    |
| Use Minio command (make bucket, copy, list...) | ✅    |
| Create a Minio cluster                         | ⏳    |
